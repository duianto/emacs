[[http://www.gfd-dennou.org/member/uwabami/cc-env/EmacsBasic.html][Emacsの基本設定]] で触れた通り,

#+begin_src emacs-lisp
  (my-load-org-file "hogehoge.org")
#+end_src

と記述するだけで

  1. hogehoge.org の begin_src emacs-lisp ... end_src の部分を hogehoge.el として抽出
  2. hogehoge.el を byte-compile
  3. hogehoge.elc を load
